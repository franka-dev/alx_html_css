# Headphones Project

This project recreates a headphone landing page based on a Figma design.

## Features

- Responsive design
- Mobile layout at 480px or below
- Hover effects on buttons and links
- Clean, centered content with max-width 1000px

## Fonts

- Source Sans Pro
- Spin-Cycle-OT

## Technologies

- HTML5
- CSS3
